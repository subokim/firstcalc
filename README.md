# 개요 (firstcalc)

이 프로젝트는 Flutter를 학습하기 위해 만들어보는 샘플 프로젝트입니다.

* 익혀야 할 언어 : Dart
* 앱개발 프레임워크 : Flutter
* 앱목표 : 간단한 계산기 앱
